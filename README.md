# Calculadora-Financeira

Projeto desenvolvido para auxiliar estudantes da área administrativa e financeira em cálculos do dia a dia, tornando processos matemáticos mais rápidos e acessíveis.
A plataforma busca simplificar o aprendizado exibindo os resultados de forma prática, juntamente às fórmula de cada cálculo. A ideia surgiu durante os estudos no curso técnico, ao perceber a dificuldade de muitos colegas em cálculos financeiros.
